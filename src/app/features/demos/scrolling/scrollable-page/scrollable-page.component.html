This page demonstrates the scroll region
Notice that the header is fixed at the top of the page and the footer is fixed at the bottom of the page, while the rest of the content scrolls. This is achieved by using grid-template-areas. The header is defined as "header" in the grid-template-areas property. The footer is defined as "footer", and the rest of the content is defined as "content". The grid-template-areas property is set to "header content" to define the layout.
<div class="grid-container">
   <div class="header">Header</div>
   <div class="content">Content</div>
   <div class="footer">Footer</div>
 </div>
.prb-main-grid-container {
   position:  fixed;
   top: 0;
   left: 0;
   width: 100%;
   display: grid;
   grid-template-areas:
      "header"
      "content"
      "footer";
   gap: 5px;

   .prb-main-grid-header {
     grid-area: header;
   }

   .prb-main-grid-content {
     grid-area: content;
     overflow-y: auto;
     padding: 10px;
     max-height:  calc ( 100 vh  -  113 px ) ;
     height: 100vh;
     background-color: #f5f5f5;
   }

   .prb-main-grid-footer {
     grid-area: footer;
   }
 }
