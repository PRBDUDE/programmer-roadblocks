Primary button should only occur once on a page. It is the primary call-to-action on the page. Secondary button can occur multiple times on a page. Tertiary button should only be used for settings or actions. Danger button should be used when there is a dangerous action about to be taken.

Note that the background color of the badge can be changed by setting the badgeSeverity. Success button should be used when the previous action was successful and you are telling the user that they can continue. Information button should be used to let the user know that there are some informative notes that they should review. Warning button is used to tell the user that there are warnings that should be reviewed before continuing.

Note that you can also note the number of warnings by using the badge. Help button is used to provide helpful information to the user. Contrasting button to use how ever you want. Plain button to use how ever you want.
